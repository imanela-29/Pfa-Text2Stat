
Pour la partie mobile, on a travaillé avec Kivy, parce que c’est une bibliothèque libre et open source pour Python et utile pour créer des applications mobiles.
Notre application a pour but de donner des statistiques qu’on les a obtenus par notre extraction des données du compte Facebook personnel du président de l’université Mohammed Premier.
Notre application n’est pas encore prête a 100% mais on a essayé de réaliser une interface qui contient le logo de notre projet , logo de l’université, un menu provisoire (Accueil, Dashboard, Mois) et la partie du connexion sur Facebook pour qu’on puisse extraire les données souhaitées.
